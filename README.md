# Netflix Content Analysis — EDA Project

An exploratory data analysis of Netflix's content library using Python.

## Overview
This project analyses 8,000+ Netflix titles to uncover trends in content type,
geography, genres, ratings, and how the platform's library grew over time.

## Key Findings
- Movies make up ~70% of the Netflix catalog
- Content additions peaked in 2019
- The United States is the top content-producing country
- TV-MA is the most common rating, showing an adult-skewed library
- International Movies is the dominant genre

## Tools & Libraries
- Python 3
- Pandas — data cleaning & analysis
- Matplotlib & Seaborn — visualizations
- WordCloud — title word cloud
- Google Colab — development environment
